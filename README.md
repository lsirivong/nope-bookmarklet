# [ NOPE ] Bookmaklet

## What is it?

A silly bookmarklet for "stamping" "NOPE" on any and all websites.

[Info Page](http://lsirivong.github.io/nope-bookmarklet)

## How do I build it?

* Install dependencies

	`npm install`

* Build

	`grunt`

This will generate a minified js, and a bookmarklet-ified js file.
